# Sandbox of td-agent

- Sandbox of td-agent

## Installed cookbooks

- td-agent
- rbenv

## How to use

```
> bundle ins --path vendor/bundle
> bundle ex berks install --path cookbooks
> vagrant up
> vagrant provision
```
